== Description ==

**Use live group chat plugin and can see all online user, only registered user can use it**

<h5><strong>Features:</strong></h5>
<ul><li>Chat is very fast and simple .</li>
<li>Using ajax messages are submitting.</li>
<li>See all online user list in left side.</li>
<li>It use database to save chat.</li>
<li>After a week chat messages will automatically deleted.</li>
<li>Simple to configure using widget and shortcode</li>
<li>On all pages we have a live chat option in footer. To use quick chat.</li>
</ul>

<h5><strong>Shortcode:</strong></h5>
<ul><li>[wp_chat_window] for showing chat window.</li></ul>

<h5><strong>Widget:</strong></h5>
<ul><li>In wordpress admin Appearance >> Widgets >> "Live Group Chat" </li></ul>

== Installation ==

1. Upload 'Wp live group Chat' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to post or page, then insert shortcode [wp_chat_window] or you can use in php file using do_shortcode( '[wp_chat_window]' );.
4. we have one more option as well, using widgets.
5. Go to Appearance >> Widgets here we have "Live Group Chat" widget.
